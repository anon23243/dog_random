SOBRE

Dog random é um aplicativo que fornece imagens aleatórias de cachorros.
Ele utiliza a API Dog.ceo para buscar e exibir as imagens.
O aplicativo foi desenvolvido com o framework Ionic/Angular, que permite criar aplicativos móveis híbridos
usando tecnologias web como HTML, CSS e JavaScript/Typescript.
